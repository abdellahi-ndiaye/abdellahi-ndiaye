## 🎨 Design

> *Where code meets creativity — I believe great software is also great design.*

I care about the full user experience: visual hierarchy, typography, color theory, and interaction design. My workflow combines wireframing in **Figma**, prototyping, and translating mockups into clean pixel-perfect frontend code.

**Design Interests:**
- `UI/UX Design` — clean interfaces, intuitive flows
- `Design Systems` — reusable components, consistent tokens
- `Typography & Color` — crafting a coherent visual language
- `Responsive & Accessible Design` — mobile-first, inclusive

## 🔐 Cybersecurity

> *Security is not a feature — it's a foundation.*

Alongside development and design, I actively explore the cybersecurity space — practicing through CTF challenges, studying network security, and applying secure coding principles in every project I build.

**Security Interests:**
- `CTF Challenges` — Capture The Flag competitions (pwn, web, crypto)
- `Ethical Hacking` — penetration testing fundamentals
- `Network Security` — protocols, firewalls, traffic analysis
- `Secure Web Dev` — OWASP Top 10, input validation, auth hardening
- `Linux & Shell` — scripting for automation and recon
